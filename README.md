# Bright Minds Tuitions

A clean, modern website for a private tuition teacher offering classes for students from Class 5 to Class 10.

🌐 **Live Site:** [https://shashivish.github.io/darshan-sharma-website/](https://shashivish.github.io/darshan-sharma-website/)

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
